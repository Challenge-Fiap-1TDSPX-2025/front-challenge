FAQ

Perguntas Frequentes

1.

Como acessar as ferramentas de ticket ou chat IA?

  • Faça login na sua conta HealthSupport
  • No painel principal, você encontrará os ícones para "Gerenciador de Tickets" e "Chat IA"
  • Clique no ícone da ferramenta que deseja acessar
  • Para o Gerenciador de Tickets, você verá todos os tickets abertos e poderá criar novos
  • Para o Chat IA, você poderá configurar respostas automáticas e monitorar interações

Todas as ferramentas estão disponíveis no menu lateral do painel de controle para fácil acesso.

2.

Como fazer login?

Para fazer login na plataforma HealthSupport:

  • Acesse app.healthsupport.com.br
  • Insira seu e-mail e senha cadastrados
  • Clique no botão "Entrar"

Caso tenha esquecido sua senha, clique em "Esqueci minha senha" e siga as instruções enviadas ao seu e-mail para redefinição.

Para o primeiro acesso, utilize as credenciais enviadas ao e-mail cadastrado durante a contratação do serviço.

3.

Como visualizar os tickets criados?

Para visualizar os tickets criados:

  • Acesse o Gerenciador de Tickets no painel principal
  • Na página inicial do gerenciador, você verá uma lista de todos os tickets
  • Utilize os filtros disponíveis para organizar por: status, data, prioridade ou departamento
  • Clique em qualquer ticket para ver os detalhes completos e o histórico de interações

Você também pode exportar relatórios de tickets através do botão "Exportar" localizado no canto superior direito da tela.

4.

Como marcar uma teleconsulta?

Para marcar uma teleconsulta:

  • Acesse o módulo "Agenda" no menu lateral
  • Clique no botão "Nova Consulta"
  • Selecione "Teleconsulta" como tipo de atendimento
  • Escolha a data e horário disponíveis
  • Selecione o paciente ou cadastre um novo
  • Adicione observações se necessário
  • Clique em "Confirmar Agendamento"

O paciente receberá automaticamente um e-mail com o link para a teleconsulta e instruções de acesso.

5.

Como personalizar as respostas automáticas do Chat IA?

Para personalizar as respostas automáticas do Chat IA:

  • Acesse o módulo "Chat IA" no painel principal
  • Clique na aba "Base de Conhecimento"
  • Você verá categorias pré-definidas como "Agendamento", "Preparos", "Documentos", etc.
  • Selecione uma categoria ou crie uma nova
  • Adicione perguntas frequentes e suas respectivas respostas
  • Configure regras de encaminhamento para atendimento humano quando necessário
  • Clique em "Salvar" para aplicar as alterações

Recomendamos revisar periodicamente as interações do chat para identificar novas perguntas frequentes e melhorar a base de conhecimento.

6.

Como integrar o HealthSupport com outros sistemas?

O HealthSupport oferece diversas opções de integração:

  • API REST para integração com sistemas próprios
  • Integrações nativas com sistemas de prontuário eletrônico populares
  • Webhooks para automação de processos
  • Integração com WhatsApp, e-mail e SMS

Para configurar uma integração:

  • Acesse "Configurações" > "Integrações"
  • Selecione o tipo de integração desejada
  • Siga as instruções específicas para cada tipo de integração

Para integrações personalizadas ou suporte técnico, entre em contato com nossa equipe de suporte.
